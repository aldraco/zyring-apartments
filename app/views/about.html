About this project

This project is courtesy of Zyring.

The purpose of this project is to demonstrate front-end skills, specifically working with Angular. The back-end data is provided courtesy Zyring.com, but the design and implementation is my own.

This project utilizes a few interesting modules/libraries:

  • Masonry
  • Google Maps API
  • UI-Bootstrap
  • ui-router

The site is mobile-friendly and aims to make the apartment-finding process a little less painless.
